安装docker
docker 配置阿里云镜像加速
/etc/docker/daemon.json

{
  "registry-mirrors": ["https://zc2hy270.mirror.aliyuncs.com"]
}

 systemctl daemon-reload
 systemctl restart docker

docker 启动命令
systemctl start docker

查看docker状态

systemctl status docker

关闭docker

systemctl stop docker

重启docker

systemctl restart docker

列出所有docker镜像
docker images

拉取镜像
docker pull 镜像名
docker pull tomcat:7

搜索镜像
docker search 镜像名

删除镜像

docker rmi 镜像名称/镜像ID

查看容器

docker ps

创建容器
docker run 创建tomcat容器

docker run --name tomcat -d -p 8080:8080 tomcat

	命令说明
		--name tomcat:给容器取个名字
		-d:后台运行
		-p 8080:8080：将容器的8080端口映射到主机的8080端口，前一个8080是主机端口，后一个8080是tomcat，
		然后可以在浏览器通过主机ip：8080访问到tomcat。第一个端口8080如果是7070，那么访问的时候就是主机ip：7070.可以更改的哦

查看所有容器
docker ps -a

查看最后一次运行的容器
docker ps -l

查看停止的容器
docker ps -f status=exited

停止正在运行的容器
docker stop 容器名称/容器ID

启动容器
docker start 容器名称/容器ID

删除容器
docker rm 容器ID


将容器打包成镜像文件
docker  commit  容器ID  镜像文件的名字（自定义）

进入容器
例如：（进入tomcat的bin目录）
docker exec -it  容器ID  bin/bash

将容器保存到tar文件
docker save tomcat >1.tar

从tar文件下载镜像文件
docker load <1.tar

复制一个目录文件到新的文件夹
cp -r webapps.dist webapps

docker 启动mysql docker容器
docker run -p 3306:3306 -e MYSQL_ROOT_PASSWORD="123456" -d mysql:5.6
